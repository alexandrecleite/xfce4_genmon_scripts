Scripts para monitoramento do sistema no painel do XFCE4 através do Monitor Genérico (xfce4-genmon).
Os tempos sugeridos de atualização estão nos comentários de cada script. São separados para poder escolher a ordem que quiser.
<BR><BR>
Se parar com o mouse em cima de cada um, aparece mais informações de cada ítem. (Tooltips)
<BR>
Se clicar com o botão esquerdo ele abre algum programa relativo ao item.
